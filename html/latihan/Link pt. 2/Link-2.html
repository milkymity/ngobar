1. Pengenalan HTML

    2. HTML

    Mengenal HTML sebagai bahasa markah yang digunakan untuk membuat dokumen. HTML adalah kepanjangan dari Hypertext Markup Language. Materi ini membahas struktur dasar HTML

  2. Heading Pada HTML

    3. Heading

    Membuat judul dan sub judul dalam dokumen HTML. Judul dan Sub judul HTML memiliki ukuran yang berbeda. Ukurannya dimulai dari level 1 sampai 6.

  3. Paragraf Pada HTML

    4. Paragraf

    Untuk membuat sebuah paragraf di HTML bisa menggunakan tag "p" sebagai awalan dan tag "/p" sebagai akhiran. Selain itu ada beberapa format untuk paragraf, seperti "b" untuk menebalkan teks dan "i" untuk teks miring. "mark" untuk menandai teks dan "u" untuk membuat teks dicoret. Dan masih banyak lagi formatting paragraf HTML.

  4. List Pada HTML

    5. List

    Dalam HTML ada 3 jenis untuk menampilkan daftar yaitu, Ordered list, Unordered List dan Definition List. Unordered List adalah daftar yang dibuat dengan simbol atau karakter di HTML. Cocok untuk membuat daftar yang tidak memiliki urutan. Ordered List adalah kebalikan dari Unordered List, yaitu sebuha daftar berurutan yang menggunakan angka, abjad atau angka romawi. Terakhir, Definition List adalah jenis daftar yang digunakan untuk mendefinisikan sesuatu.

  5. Link Pada HTML

    6. Link

    Link berfungsi untuk berpindah ke halaman web yang sama atau web yang berbeda. Link pada HTML memiliki 2 tipe, Internal dan Eksternal. Internal berguna untuk menuju ke halaman lain di web yang sama. Sedangkan Eksternal berguna untuk ke halaman web yang berbeda. Atribut pada Link ada "Title" dan "Target" yang berfungsi untuk memberi judul link dan menentukan dimana web akan di buka.

  6. Image Pada HTML

    7. Image

    Menyisipkan gambar ke dalam sebuah web dengan 2 cara, yaitu Internal dan Eksternal. Gambar internal berada pada file yang sama. Sedangkan eksternal mengambil gambar dari situs luar seperti Google & Pinterest. Tag "width" untuk mengatur lebar gambar dan tag "height" mengatur tinggi gambar.
